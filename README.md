# Minecraft Seed Map JP v5.1

Minecraftのシード値から、疑似バイオーム、スライムチャンク、構造物候補、手動マーカーを確認できる日本語の静的Seed Mapです。

v5.0でCanvasベースのマップへ移行し、v5.1では操作性と描画パフォーマンスを改善しました。

## v5.1の更新内容

- Canvas描画を `requestAnimationFrame` ベースに調整
- ドラッグ中は簡易描画にして、停止後に詳細描画
- 疑似バイオーム描画結果をキャッシュ
- 表示範囲外の構造物候補/手動マーカーを描画対象外にする
- ズーム段階を拡張
  - 広域
  - 広域+
  - 標準
  - 詳細
  - 近距離
- ズーム倍率表示を追加
- 0,0を「初期地点」としてマップ上に表示
- 「中心地 0,0」レイヤーON/OFFを追加
- ドラッグ中のカーソルを `grabbing` に変更
- ホイールズームはマウス位置を基準に拡大縮小

## 主な機能

- Java版/統合版のエディション選択
- バージョン選択
- 疑似バイオーム表示
- スライムチャンク表示
- 構造物候補表示
- 地点メモ/手動マーカー
- 座標ジャンプ
- 座標コピー
- ネザー座標変換
- レイヤーON/OFF
- カテゴリフィルタ

## 構造物候補

以下の構造物を候補として表示します。

- 村
- 要塞
- 廃ポータル
- 海底神殿
- 森の洋館
- ピリジャー前哨基地
- 古代都市
- トライアルチャンバー

構造物は候補表示です。実際のMinecraft生成位置やChunkbaseの表示と完全一致する保証はありません。

## 詳細バイオームについて

現時点では疑似バイオーム表示です。正確なJava版バイオーム判定や構造物生成に近づけるには、将来的にcubiomes WASM統合が必要です。

- cubiomes本体は同梱していません
- `assets/wasm/` は将来のWASM配置先です
- 詳細バイオームモードは準備中です
- WASM未配置でもアプリは壊れず、疑似バイオームで動作します

## ローカル実行

ES Modulesを使っているため、プロジェクトルートで静的サーバーを起動して確認してください。

```powershell
python -m http.server 8000
```

ブラウザで以下を開きます。

```text
http://localhost:8000/
```

## 動作確認ポイント

- マップをドラッグで軽く移動できる
- ホイールで以前より深くズームできる
- ズーム倍率が画面上で確認できる
- 0,0初期地点がマップ上で分かりやすい
- seed変更で疑似バイオームが変わる
- Java/Bedrockの両方で構造物候補が表示される
- スライムチャンクOFFでも構造物候補が表示される
- 手動マーカー、座標コピー、ネザー変換が動く
- 通常操作でコンソールエラーや頻繁な `console.warn` が出ない

## 保存と通信

- 外部通信なしで動作します
- ユーザー入力はブラウザ内の `localStorage` 以外に保存しません
- GitHub Pagesで公開できる静的構成です

## 注意

このツールはChunkbaseの完全な代替ではありません。v5.1時点では、操作感と見た目をSeed Mapらしく近づけることを優先し、構造物とバイオームは候補表示・疑似表示として扱っています。
