# Minecraft Seed Map JP v1.1

Minecraft Java版のシード値から、スライムチャンクを日本語UIで確認する静的Webツールです。v1.1はJava版スライムチャンク判定と探索支援機能に対応しています。外部API、外部ライブラリ、アクセス解析は使用していません。

## 主な機能

- 数値または文字列のシード値入力
- Java版のスライムチャンク判定（v1.1は統合版未対応）
- Minecraftバージョン選択
- 中心X/Z座標と表示範囲の指定
- チャンク単位のスライムチャンクグリッド表示
- 選択チャンクのチャンク座標、ブロック範囲、中心ブロック座標、判定表示
- 選択チャンク詳細と中心ブロックX/Z座標のコピー
- オーバーワールド座標とネザー座標の相互変換
- 探索メモの追加、表示、削除、全削除
- スマートフォン幅に対応したレスポンシブ表示

## セットアップ

追加インストールは不要です。リポジトリを取得し、`index.html` をブラウザで開くだけで動作します。

```text
minecraft-seed-map-jp/
├── index.html
├── README.md
├── css/
│   └── style.css
├── js/
│   ├── main.js
│   ├── slime.js
│   ├── storage.js
│   └── utils.js
└── assets/
    └── icons/
        └── favicon.svg
```

## ローカル実行

このプロジェクトはビルド不要です。基本的には `index.html` をブラウザで開くだけで確認できます。

ブラウザのセキュリティ制限でJavaScriptモジュールが読み込めない場合は、プロジェクトのルートディレクトリで静的サーバーを起動してください。

```powershell
python -m http.server 8000
```

その後、ブラウザで `http://localhost:8000/` を開きます。

## 動作確認ポイント

ローカルで開いたら、以下を確認してください。

1. `index.html` を開くと、条件入力フォーム、チャンクグリッド表示エリア、探索メモ欄が表示される。
2. シード値、中心X/Z、表示範囲を入力して `マップを生成` を押すと、チャンクグリッドが生成される。
3. グリッド上のチャンクをクリックすると、チャンク座標、ブロック範囲、中心ブロック座標、スライム判定が表示される。
4. `詳細をコピー` と `X/Zをコピー` で、選択中チャンクの情報をコピーできる。
5. ネザー座標変換で、オーバーワールド座標とネザー座標を相互変換できる。
6. 探索メモを追加すると一覧に表示され、削除や全削除ができる。
7. ページを再読み込みしても、探索メモがブラウザの `localStorage` に残っている。
8. 画面幅を狭くしても、フォームやグリッド周辺の表示が大きく崩れない。

## GitHub Pagesでの公開

1. GitHubに `minecraft-seed-map-jp` リポジトリを作成します。
2. このディレクトリの内容を `main` ブランチへ push します。
3. GitHubの `Settings` → `Pages` を開きます。
4. `Build and deployment` を `Deploy from a branch` にします。
5. Branchを `main`、Folderを `/root` にして保存します。

## 使い方

1. シード値を入力します。
2. エディション、バージョン、中心座標、表示範囲を選びます。
3. `マップを生成` を押します。
4. グリッド上のチャンクを選択すると、チャンク座標とブロック範囲が表示されます。
5. 必要に応じて `詳細をコピー`、`X/Zをコピー`、ネザー座標変換、探索メモを使います。

## 入力項目

- シード値: 数値または文字列。文字列はJavaの `String.hashCode` 相当で数値化します。
- 中心X/Z: ブロック座標の整数値。
- 表示範囲: 16、32、64チャンクから選択します。

## 出力内容

- チャンクグリッド: スライムチャンクは緑、通常チャンクはグレー、中心付近は枠で強調します。
- 選択チャンク詳細: チャンク座標、ブロック座標範囲、中心ブロック座標、スライム判定。
- ネザー座標変換: オーバーワールド座標とネザー座標のX/Z変換結果。
- 探索メモ: タイトル、X/Z座標、種別、本文。

## セキュリティとプライバシー

このツールは外部通信を行いません。入力されたシード値、座標、メモ本文は外部サーバーへ送信されません。

探索メモはブラウザの `localStorage` にのみ保存されます。共有PCや公開端末で使う場合は、使用後に画面内の `全削除` を押すか、ブラウザのサイトデータを削除してください。

## 注意事項

- このツールは非公式のMinecraft補助ツールです。
- MinecraftおよびMojang Studiosとは関係ありません。
- v1.1のスライムチャンク判定はJava版のみ対応です。
- 統合版はUI上で選択できますが、v1.1ではスライムチャンク判定を行いません。Java版を選択してください。
- 村、要塞、廃坑、バイオーム、地形、3D表示、ログイン、外部API連携、広告、アクセス解析、PWAには対応していません。
- 構造物の自動検出は実装していません。
