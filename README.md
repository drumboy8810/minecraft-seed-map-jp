# Minecraft Seed Map JP v1.0

Minecraft Java版のシード値から、スライムチャンクを日本語UIで確認する静的Webツールです。外部API、外部ライブラリ、アクセス解析は使用していません。

## 主な機能

- 数値または文字列のシード値入力
- Java版 / 統合版の選択（v1.0のスライム判定はJava版を優先）
- Minecraftバージョン選択
- 中心X/Z座標と表示範囲の指定
- チャンク単位のスライムチャンクグリッド表示
- 選択チャンクのチャンク座標、ブロック範囲、判定表示
- 座標コピー
- 探索メモの追加、表示、削除、全削除
- スマートフォン幅に対応したレスポンシブ表示

## セットアップ

追加インストールは不要です。リポジトリを取得し、`index.html` をブラウザで開くだけで動作します。

```text
minecraft-seed-map-jp/
├── index.html
├── README.md
├── css/
│   └── style.css
├── js/
│   ├── main.js
│   ├── slime.js
│   ├── storage.js
│   └── utils.js
└── assets/
    └── icons/
        └── favicon.svg
```

## ローカル実行

そのまま `index.html` を開いてください。モジュール読み込みが制限されるブラウザ環境では、任意の静的サーバーでルートディレクトリを配信してください。

```powershell
python -m http.server 8000
```

その後、ブラウザで `http://localhost:8000/` を開きます。

## GitHub Pagesでの公開

1. GitHubに `minecraft-seed-map-jp` リポジトリを作成します。
2. このディレクトリの内容を `main` ブランチへ push します。
3. GitHubの `Settings` → `Pages` を開きます。
4. `Build and deployment` を `Deploy from a branch` にします。
5. Branchを `main`、Folderを `/root` にして保存します。

## 使い方

1. シード値を入力します。
2. エディション、バージョン、中心座標、表示範囲を選びます。
3. `マップを生成` を押します。
4. グリッド上のチャンクを選択すると、チャンク座標とブロック範囲が表示されます。
5. 必要に応じて `座標をコピー` や探索メモを使います。

## 入力項目

- シード値: 数値または文字列。文字列はJavaの `String.hashCode` 相当で数値化します。
- 中心X/Z: ブロック座標の整数値。
- 表示範囲: 16、32、64チャンクから選択します。

## 出力内容

- チャンクグリッド: スライムチャンクは緑、通常チャンクはグレー、中心付近は枠で強調します。
- 選択チャンク詳細: チャンク座標、ブロック座標範囲、スライム判定。
- 探索メモ: タイトル、X/Z座標、種別、本文。

## セキュリティとプライバシー

このツールは外部通信を行いません。入力されたシード値、座標、メモ本文は外部サーバーへ送信されません。

探索メモはブラウザの `localStorage` にのみ保存されます。共有PCや公開端末で使う場合は、使用後に画面内の `全削除` を押すか、ブラウザのサイトデータを削除してください。

## 注意事項

- このツールは非公式のMinecraft補助ツールです。
- MinecraftおよびMojang Studiosとは関係ありません。
- v1.0のスライムチャンク判定はJava版向けの簡易判定です。
- 統合版はUI上で選択できますが、v1.0では正確なスライムチャンク判定を提供しません。
- 村、要塞、廃坑、バイオーム、地形、3D表示、ログイン、外部API連携、広告、アクセス解析、PWAには対応していません。
